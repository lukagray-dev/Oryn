# Oryn

</div>

A minimalist WYSIWYG markdown editor. Write structured documents — scripts, notes,
outlines — without touching markdown syntax.

Built for creators and note-takers who want the speed of markdown without the
learning curve: no `**`, no `#`, no syntax to remember. Type, and it formats itself.

## Status

> Early development (v0.1.0). Not yet ready for use.

## Architecture

- `oryn-rs/` — core document model, markdown parsing/serialization. Shared across
  all frontends.
- `gui/` — desktop UI, built with [Slint](https://slint.dev).

A single Rust backend powers desktop and (planned) mobile frontends.
